Class Allocator

User Allocator

Used to assign a pool of users to specific roles inside of a work flow. See previous work http://web.njit.edu/~mt85/UsersAlg.php

To run an intelligent version of the allocator, you should check out the Drupal\ClassLearning\Workflow\Allocator::assignmentRun() method. It will run the allocation a few times to ensure there are no errors.
